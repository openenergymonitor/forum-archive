EmonCMS Developer Meetup and Discussion Thread

Submitted by XaroRSA on Wed, 04/09/2013 - 08:37

Good day Forum.

I would like to make a thread for discussion regarding Development of EMONCMS.

I have been trying to add multiple features, such as extra graphs, widgets, data generators etc, but the best I have done is some hacked up version that barely works.

Some issues that I have with EMONCMS is the lack of documation and examples. The code is not commented and newbies such as myself have a hard time understanding the implementation of the coding.

My suggestion to greatly improve the OpenEnergyMonitor project:

We have a online meetup with interested developers. We work on basic, and just understanding of how the program runs. I believe that if we can get some basics, such as adding a widget properly into the current system, we can have alot more people(such as myself) contributing alot more to the front-end of the system, while main contributors work on core functionallity such as timestore etc.

Of course this is just a suggestion and this is a open project so no-one is obligated to provide support or make it better, but I firmly believe with just a little tweak the project can be boosted into something amazing.(its already amazing :D)

Let me know your thoughts on this and if im completely wrong?

Regards
