Revisions to EmonLib - 7 June 2012

The variable "wavelengths" in the library functions has been renamed to "crossings" in order to more accurately reflect its purpose. The new version is available on Github.

This change does not affect how the routines work.

Ideally, the number chosen should be an even number so that a whole number of voltage waves, and the corresponding current waves, are measured.
