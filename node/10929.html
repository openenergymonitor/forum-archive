EmonCMS inputs not updating

Submitted by hendersoniain on Thu, 02/07/2015 - 20:46

Hi,

 

We had a power cut last night. When the power came back on this evening I started my Pi normally and all seems to be running fine. However, none of my inputs update. Data is coming from nodes OK (EmonGLCD see's all the data). There is nothing appearing in the emonhub.log since the power failure last night. I have rebooted, restarted emons (which says it restarts OK and the process is running) but nothing works. Emoncms version is 8.40. I have had plenty power cuts before and everything has always started up OK in the past. I have tried looking through the forums but cannot see anything similar. Any ideas?

Thanks.

Re: EmonCMS inputs not updating

Submitted by hendersoniain on Fri, 03/07/2015 - 20:11.

Hi,

Doh! Forgot to set to DEBUG level but I just found the problem this evening. NTP hadn't updated time on the Pi since the power went off. Once I restarted the NTP service and the time corrected itself then the inputs started coming OK. Not sure why having the wrong time in the Pi stopped the emoncms working locally but that is what has fixed it.

Thanks for reminding me about the DEBUG level.
