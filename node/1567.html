Re: History graph for the temperature

Submitted by ukmoose on Fri, 07/12/2012 - 10:37.

The attributes of the Feed will determine what type of graph it makes sense to display the data on. 

 

If you click on Feed, find the ID of the feed you are trying to display.

If the feed is a temperature reading taking at a point in time, the datatype  should read REALTIME if it doesn't click on the pencil to edit the line and change it to REALTIME.  Click on the tick to save data in that line.

Now click on Vis.

1) Select visualisation "realtime"
2) Set options to the feed you are trying to display.
3) Click on View

If this still displays nothing, try downloading a new web-browser and see if you get the same in that.
