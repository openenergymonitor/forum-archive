Sounds like you need to open up the samba ports on the Pi's firewall. Can't remember the exact syntax to use but google will help. From mine I have

pi@emonpi ~ $ sudo ufw status
Status: active

To                         Action      From
--                         ------      ----
80/tcp                     ALLOW       Anywhere
443/tcp                    ALLOW       Anywhere
22/tcp                     ALLOW       Anywhere
135/tcp                    ALLOW       192.168.0.0/16
137/udp                    ALLOW       192.168.0.0/16
138/udp                    ALLOW       192.168.0.0/16
139/tcp                    ALLOW       192.168.0.0/16
