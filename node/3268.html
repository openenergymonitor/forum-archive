Re: Setting up a 150 kW solar system, can I use emonTx?

Submitted by Robert Wall on Thu, 05/12/2013 - 14:07.

"as each CT sensor is 23 kW max?" That is only true of the standard shop c.t. You don't have to use that, you can use any other appropriately rated c.t. The software limit (if you send power in watts which is the default) is 32767 W. There's no reason, given that size of installation, to use watts as the base unit, you could use "decawatts" or "hectowatts" for example (1 HW = 10 DW = 100 W) and appropriately change it for display and logging.

I need a full line diagram to give you any detailed advice.
