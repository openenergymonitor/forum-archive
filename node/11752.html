Re: git checkout 9.x

Submitted by Paul Reed on Sat, 05/12/2015 - 19:53.

In /etc/init.d can you see a symlink called 'feedwriter'?

If so delete it and run:

sudo update-rc.d feedwriter remove

If not - that's OK

In both cases then reinstall the feedwriter init script & reboot.

cd /etc/init.d && sudo ln -s /var/www/emoncms/scripts/feedwriter
sudo chown root:root /var/www/emoncms/scripts/feedwriter
sudo chmod 755 /var/www/emoncms/scripts/feedwriter
sudo update-rc.d feedwriter defaults

Paul
