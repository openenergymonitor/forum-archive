Re: Anyone got a new Raspberri Pi B+ working?

Submitted by pb66 on Thu, 18/12/2014 - 20:48.

Hi Walter,

The issue Bill had was due to emonhub not seeing the correct settings as they were being edited in the wrong location. On the "13/08/14 image" the location is /boot/emonhub.conf where as the default location is /etc/emonhub/emonhub.conf.

Easy check to see if you have the right one is to set "loglevel = debug" in the conf and check the log to see if it has changed using

tail -f /var/log/emonhub/emonhub.log

There shouldn't be any issues running on a B+, let us know if you are still having an issue after checking the conf and log. (post a bit of the log maybe)

Paul
