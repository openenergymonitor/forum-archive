Elektrocamp 25/26 April 2014 Leuven Belgium

Submitted by TrystanLea on Mon, 31/03/2014 - 11:51

The next ElektroCamp date has been set! ElektroCamp is a great event that brings together people interested in an open source approach to Smart Metering, Smart Home, Smart Grid usually from all over europe. It is a participative workshop ran in BarCamp style. I've been to a couple of these so far and they are really good fun and worth the trip.

The next elektrocamp is in Leuven, Belgium, in just over 3 weeks 25/26th of April. It would be great to meet people if you can come!

The event sign up and more info can be found here: http://developer.mysmartgrid.de/doku.php?id=ec1404-coordination
