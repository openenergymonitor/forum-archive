Re: What would I need in a 3 phase instalation to monitor:

Submitted by Robert Wall on Tue, 17/06/2014 - 22:11.

As far as I can recall, no-one has published a 3-phase sketch for the GLCD, so you will need to write your own sketch (more likely: modify an existing sketch) to show the 3-phase information. There is a description of the "standard" sketch here: https://github.com/openenergymonitor/EmonGLCD/blob/master/SolarPV/SolarP... which should help you. If you have difficulties, you can get help in these forums.

You can use the same emonCMS, or you can make a fresh start. If you are using emoncms.org, you might want to create a new account. If you have emonCMS on your own server, you might find it easier to modify it to handle the extra information.
