Re: emon v2 or v3 for 3 phase and PV monitoring

Submitted by Robert Wall on Thu, 29/10/2015 - 00:10.

The main difference between the V2 and V3 (apart from the V2 being no longer available as a kit) is the V2 has 3 current inputs (but it's relatively easy to add another two using some prototype board externally), and the V3 has four current inputs. We believe the V3.4 is a little quieter electrically so the performance is maintained to slightly lower powers than with the V2.

Re: emon v2 or v3 for 3 phase and PV monitoring

Submitted by Robert Wall on Sun, 15/11/2015 - 23:23.

Thanks, I understand now. With what is essentially a separate installation for each phase, then you don't and can't account for the phase displacement between the phases. But of course you'll only be able to measure real power on phase 1 (the one with the ac adapter), and you'll need a 5 V dc supply for the emonTx's on phases 2 & 3.

But it looks as if the CTs are the secondary measurement anyway, because by definition the kWh optical input from your supplier's meter is correct (even if it's wrong!), and I assume you'll be using the CTs principally for measurement of the peak load current, and of course you'll get the load distribution across the phases as well. That could be important when it comes to in-feed tariffs and stuff like that, because you might find that you get charged for import on one phase even if you're nett exporting but most of the export is on the other two.
