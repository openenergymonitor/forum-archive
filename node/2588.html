Re: 500A CT Sensor

Submitted by Robert Wall on Mon, 15/07/2013 - 22:27.

I don't have a specific recommendation. Your c.t. will need to have a sufficiently large VA rating to be able to generate about 1 V rms across the burden resistor, if you are using the standard emonTx configuration that uses the 3.3 V supply as the analogue reference. If you change to using the internal 1.1 V reference, you should be able to use the much more common 0.333 V output c.t's, but your sketch will need changing from the examples here
