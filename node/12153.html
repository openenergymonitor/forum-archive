How about writing data to ziinode.io cloud

Submitted by ziinode on Tue, 09/02/2016 - 10:10

Hello,

Im working for some time on cloud based storage and dashboard engine ziinode.io and want let to know someone out there about it and get some feedback.

Why cloud?, well, because wanted to create solution where you can just connect device to internet and see everything in one place even if devices are in different locations. Also there are benefit for you as a developer, you can earn twice - one time when you selling device and later when user pays to cloud for device usage - you get for example 10% of user pays to cloud.

Posted on github code sample for various platforms on github https://github.com/ziinode Just launched this project and there are still lots of work on documentation and could be some bugs still to fix, some feedback is welcome.

Why this form? Because I am great fan OpenEnergyMonitor project, also tried some things from OEM on arduino platform and considering to add current sensor capability to our board http://ziinode.io/boards.html  

There are dashboard demo link http://ziinode.io/graf/apiz?server=011rix.ziinode.io&apik=a1f46ff0-c8c2-... this is without administration capabilities, but if you want to try out more, you are welcome to register account and play around.
