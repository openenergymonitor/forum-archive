The raspberrypi module was put out to pasture sometime ago along with the oemgateway and a couple of methods of hooking up an rfm2pi. The recommended method now is emonhub.

There is a "sticky" on one of the forum sections about installing emonhub, there is a script specifically for transitioning from the raspberrypi module to emonhub mentioned there.

There are links to the repo, a step by step guide by Paul Reed and some discussion about using the script.

The latest emoncms install guide (also by Paul Reed) also quotes a one-liner command to install emonhub as part of the local emoncms install, but that one does not remove the raspberrypi module.

There is also another "emon-pi" variant included in the ready made sdcard image for read-only SDcard installs on the Pi too.

Paul
