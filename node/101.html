LCD Power Meter

by Matt Hodge

This build incorporates a LCD, a Pololu ACS715, a Diecimila and an Adafruit SD logger shield.

Update February 2011

Matt recently update me with his latest developments in his own words:

I have replaced the Diecimilanove with a clone of an Australian Arduino clone (Freetronics 2010, www.freetronics.com). It addresses a few of the Diecimilanove bugs eg. LED 13 loading and throws in a few extras eg. 3V3 regulated supply!

Have built a LCD pcb driver board using the Microchip MCP23017. This replaces the original veroboard version used in the initial design. Have also built a 16 channel, 10bit ADC (can be expanded to 96 channels with multiple boards). This is based on the JeeLabs Input Plug V2. Have found this handy for logging analogue sensor data.

When I get the time I intend to build a pcb to replace the veroboard energy meter board.
Anyway, I'm attaching the details of the design modifications, someone might find this of interest.
