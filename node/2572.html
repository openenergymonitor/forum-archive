Re: Replicating Dashboards

Submitted by EnergyRnR on Thu, 11/07/2013 - 06:09.

You can 'clone' dashboards in the dashboard 'list' view very easily, so this sounds like what you need to do. i.e. create the dashboard and then clone it 33 times.

Alternatively, if the dashboards are not in the same emoncms server, you could create one dashboard and then 'view source' in the browser and past the html definitions you need directly into the DB through phpmyadmin.
