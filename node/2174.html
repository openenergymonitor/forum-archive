Connecting RFM12Pi expansion board to Raspberry pi

Submitted by Guest on Fri, 29/03/2013 - 19:46

Just finished soldering my RFM12Pi expansion board together and I'm not sure how to connect it the the Raspberry Pi.

On the Raspberry Pi there is a 26 pin connector, where as the RFM12Pi board has a 10 pin connector. I'm guessing from the pictures that it is  Pin one to Pin one of each unit, but cannot see that documented any where and don't want to blow anything up by connecting it up wrong.

Can anyone help me out?

TIA

Joel

Re: Connecting RFM12Pi expansion board to Raspberry pi

Submitted by Paul Reed on Fri, 29/03/2013 - 19:58.

Hi Joel, yes you are correct, so that both boards line up with each other on the edge side (nearest the SD card).

Make sure that you engage both row of pins. It's easy to clip the board on just connecting one row only by mistake.

Paul
