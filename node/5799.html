Emoncms v8.3.6

Submitted by TrystanLea on Fri, 19/09/2014 - 17:20

A brief update, I've marked the current emoncms (full-version master branch) state with a new version tag version 8.3.6

It includes a series of bug fixes over version 8.3.5 mostly to do with the visualisations the changelog can be viewed here in a convenient github auto generated form here: https://github.com/emoncms/emoncms/compare/v8.3.5...v8.3.6

The recent commits to the master branch of emoncms still need to be copied over to the low write version. So the bugs found in the master branch will likely still be there on the low write SD card version until this is done.

I've also started to compile a development plan overview here, summarising developments from earlier this year and ideas going forward https://github.com/emoncms/emoncms/issues/244
