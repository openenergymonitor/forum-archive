Submitted by mkrnak on Mon, 28/12/2015 - 09:04

Hi to all.

After upgrade my local Pi running emoncms from HD to emon 9.2 version I am facing problem with showing daily charts especially the zoom chart is wrong and simple zoom is wrong partly.

For simple zoom - all works fine 00:00 - 12:00, after this there is issue wit current day - I think that current day shows data from from different time shift. The value is usually much bigger than value in Myelectric app, which seems to be correct.
