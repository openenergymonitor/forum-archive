Submitted by ukmoose on Mon, 26/05/2014 - 12:10.

It really depends on what you are using as your emonBase (or base station)

If you are planning on using the  NanodeRF you will see it contains an ethernet port which is the easiest way of linking it up.

If you want to use wifi, you are probably better using a raspberry pi and a RFM12Pi cards and buying a wifi usb dongle for the Raspberry Pi.
