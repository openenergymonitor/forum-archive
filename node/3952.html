Re: temperature

Submitted by ukmoose on Sat, 01/03/2014 - 09:49.

I think it will NOT work with the SparkCore library -

Please follow these instructions:

  1. Delete DallasTemperatureSparkCore Library.
  2. Download this one :  http://download.milesburton.com/Arduino/MaximTemperature (get the LATEST.zip)
  3. Install it into Arduino/libraries (you will need to change the folder name to not have - in)
  4. Quit Arduino IDE (if it is open)
  5. Open Arduino IDE
  6. Try again
