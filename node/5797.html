Yes. Something like that worked for me.

EDIT: you normally do not have to 'add' any nodes there because all data coming in from the RFM12Pi board contains the node information already and the inputs appear automatically.

 

But I had to find out the hard way that there are two different installation guides. If you follow the guide here:

https://github.com/emoncms/emoncms/blob/bufferedwrite/docs/setup.md 

the system will not autostart and some other things will not work, too.

Paul (pb66) showed me this link with some extra settings which leads to a fully working, autostarting system:

https://github.com/emoncms/emoncms/blob/bufferedwrite/docs/install.md
