Submitted by pb66 on Thu, 13/02/2014 - 13:06.

Using putty you can change read only access to read and write using 

   ipe-rw

at the command-line, edit using

   nano /boot/oemgateway.conf

and then use

   ipe-ro

to make it read only again.
