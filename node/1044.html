You should just need to open Eagle, then click on file>open>board then select the emontx.brd file. You shouldn't need to create a new project. I've never had that error before. You will need to use Eagle 6.2 or later.

I had exactly the same problem when first trying to download/open the file. This happens if you right click the blue part in 'path: root/EmonTx_v2.2.brd' and save this. File size is then ~1.4Mb.

I had to right click the blue plain in 'blob: 30254e46b13c0fe53eb33fad5c112243316780a4 (plain)' to download the correct file with ~220k file size.

Hope this helps.
