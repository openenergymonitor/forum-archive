Re: Generate random feed data and commercial use

Submitted by Robert Wall on Tue, 27/08/2013 - 10:10.

You should be able to post data using a combination of the examples in the Input API help page. Any script would simply automate that.

(I've never needed to, I just programmed an emonTx to generate the numbers and run them through the whole system, so testing it end to end, so to speak.)

Regarding commercial use, my personal view is if a business is making use of the service, then the honourable thing for the business to do is to make a regular contribution so that emoncms and indeed this website can continue to operate. After all, if the business had to buy, run and maintain its own server and buy in technical expertise to replace the help you get from this website, there would be a cost; and it's that cost that the business is saving. I fully realise that bean-counters might not see it that way.
