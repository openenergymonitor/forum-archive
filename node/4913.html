Double click needed to make feeds public

Submitted by pb66 on Fri, 04/04/2014 - 22:53

Another little quirk of emoncms v8 is when clicking the briefcase to change it to a globe and make a feed public, it requires a rapid double click if the briefcase is currently displayed and a single click if the globe is displayed. it would seem the same "false" json command is used each click regardless of current state, only a rapid double click will issue the required "true" json as the second command needed to switch from private to public.

I've compared this to emoncms.org which I found just one click does the job everytime. So I don't think the double click is meant to be there but now I am aware I can work with it.

See the screenshot of a chrome dev tools window, you can see the initial command is always false.

Paul
