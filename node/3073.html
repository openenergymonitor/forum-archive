Re: Emoncms values

Submitted by TrystanLea on Sun, 03/11/2013 - 07:57.

Ah, in adding redis to save the last feed time and values rather than mysql i missed adding it to the public feeds function, its now added and the values changed on the public dashboard ok for me.

To fill in the missing day, its a bit fiddly but you need to enter a timestamp for the day you want to create.

The timestamp for the 29th is 1383091200 and the 31st is 1383264000 you can check that its exactly 2x 86400s (where there are 86400s in a day).

Then add 86400 to 1383091200 = 1383177600

place that timestamp in the time box and a new value in the new value box, click ok and it should create it.
