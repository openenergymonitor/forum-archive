Hi Sunanda,

As far as I can see the only difference is India uses 220V instead of 240V. If you use the emonTx with an AC-AC adapter no change is required. If you are using just a CT you will need to change the AC RMS voltage in the sketch see line 83-93 : https://github.com/openenergymonitor/emonTxFirmware/blob/master/emonTx_CT123/emonTx_CT123.ino. 

The cost per unit of electricity can easily be set in emoncms.
