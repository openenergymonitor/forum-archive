In case someone gets interested in the task, I've checked and there is a bit more than 70 terms spread in around 15 files that would need to be ported. I've got those numbers by running a python script http://pythonhosted.org//b2tob3/ (with a small modification to include php files) that does the conversion (though one is supposed to use versioning control and review the changes) and outputs the nb of changes.

As expected, the simple running of the python script resulted in some weird layout and broken links. I dont feel like I know the code or bootstrap enough to do the migration, but I hope the notes here can be of any help.
