Re: Emoncms.org site doesn't support iframe anymore [SOLVED]

Submitted by maxveneziano on Thu, 14/04/2016 - 12:52.

SOLVED

The problem is due to the used browser setting and not to emoncms site.

Firefox by default is set to block "mixed contents" when you use an https request (emoncms site works with https protocol). To unblock it just click on hit use an http protocol)

On other browser you can set it a in slightly different way (IE, Chrome)
