Design for gas meter reader using optical mouse chip

Submitted by daveb on Tue, 07/07/2015 - 13:54

Hi All

I thought you'd be interested in my design for a gas meter reader using an optical mouse chip.

http://www.rotwang.co.uk/projects/gasmeter.html

It uses an Arduino Nano connected to an ADNS2610 to image the dial on a mechanical gas meter. I developed some image processing code to resolve the angle of the dial. It seems to work very well.

Comments welcome.
