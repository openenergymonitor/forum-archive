Actually, if you haven't sent back the emonPi yet. Could you do a quick debug? It might be an easy issue to fix.

Now the emonPi has been removed from the Raspberry Pi does the red LED on the emonPiu light up then turn off in a few seconds when power is applied via the mini-USB connector, without being connected to the Pi. If so the emonPi is working fine. The issue could be the OOK module PCB footprint pads shorting out on top of the on Pi's USB sockets. There should be a layer of tape present on top of the Pi's USB sockets to insulate against this. This issue will be fixed on the PCB rev
