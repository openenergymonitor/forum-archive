Submitted by glyn.hudson on Thu, 08/10/2015 - 10:37.

The new pulse counting firmware should work fine with a reed switch: https://github.com/openenergymonitor/emonTH/tree/master/emonTH_DHT22_DS18B20_RFM69CW_Pulse

Connect in the same way as a pulse sensor: http://wiki.openenergymonitor.org/index.php/EmonTH_V1.5#Pulse_Sensor_Connection

This new firmware will be shipped as standard. We have finished doing a bit more testing. For the moment, you will have to upload it yourself via the Arduino IDE, and a USB-to-UART cable. If you are ordering via the shop, leave a note at checkout. We can add the new pulse firmware for you, if you like.
