1) the favourite dashboard icon is off the left of the screen in portrait mode on my iPhone 6, it does appear in landscape though.

Because of the limited screensize on small devices, certain things are changed/hidden via css, and the 'favourite' dashboard is not displayed by design.

2) changes to dashboards can't be easily saved as the save button is behind the menu bar. Switching to landscape and really zooming in allows the edge of the save button to be hit

I can see the whole save button only in landscape on a iphone 6, again because the screen size is so small it has limitations. However good luck Kevin laying out dashboards on a 4.7" screen, you're a braver man than me!
