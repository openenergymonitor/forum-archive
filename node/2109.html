Solar PV Volts & Amps monitor integration to emon (Which regulator to use ?)

Submitted by Petrik on Sat, 09/03/2013 - 10:36

 

Just built a small device to monitor the PV generated volts and amps to transmit the PV volts and amps wirelessly to emoncms and to emonGLCD. The core is jeenode with ACS758 current sensing device. Used Tx_SimpleRFM12B_demo script and it was like a 5 minute job to get it up and running which proves how versatile the Emon environmentn.

Instead of having an external power supply to provide power to jeenode started to wonder if there is a suitable wide range regulator up to 50V that could be found so that the PV cells would power up the jeenode. The current jeenode operating range for 3.3v regulator is way too narrow to be used with a PV cell. (24V system with MPPT voltage 2x17V=34V).

Any suggestions which regulator would be capable of handing input voltages up to 50V and then provide 5V or 5-12V output range.
