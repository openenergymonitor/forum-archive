[SOLVED] On emoncms.org, how do I set a feed or a multigraph to be publicly viewable?

Submitted by gluino on Mon, 03/11/2014 - 10:15

I have an emoncms (free) account, and I have a few emonTH's sending updates once every 5 minutes, the feeds' interval are also set at 5min.  I've figured out how to use Multigraph, works fine.

Now I would like to be able to set the whole thing as publicly readable/viewable, but I can't find a way.

Thank you.

UPDATE: SOLVED

Set a feed to public by clicking the little padlock icons on the "Feeds" page of emoncoms.org .   Once the feeds are set public, the Multigraph using those feeds, will also be publicly viewable, i.e. for non-logged in browsers.

Re: [SOLVED] On emoncms.org, how do I set a feed or a multigraph to be publicly viewable?

Submitted by gluino on Tue, 04/11/2014 - 01:36.

Oops, sorry, I found the "Public" column on the "Feeds" page of http://emoncms.org/ when logged in. There are little icons of padlocks, click each one to toggle to "globe" icon. I set all my feeds to the globe icon, and now my multigraph is viewable via  the "fullscreen" url : ( http://emoncms.org/vis/multigraph?mid=NNNN ) from an incognito browser, non-logged-in.

I'll mark this as solved.
